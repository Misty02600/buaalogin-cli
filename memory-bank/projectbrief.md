# Project Brief: buaalogin-cli

## 项目概述

buaalogin-cli 是一个 BUAA（北京航空航天大学）校园网自动登录与保活的命令行工具。基于 Playwright 浏览器自动化技术实现，提供开箱即用的校园网登录解决方案。

## 命名规范

| 项目         | 值              | 说明               |
| ------------ | --------------- | ------------------ |
| **包名**     | `buaalogin-cli` | pip install 使用   |
| **模块名**   | `buaalogin_cli` | Python import 使用 |
| **CLI 命令** | `buaalogin`     | 终端执行命令       |
| **APP_NAME** | `buaalogin-cli` | 配置/日志目录名    |

## 核心需求

1. **自动登录**：模拟浏览器登录校园网认证页面
2. **持续保活**：定期检测网络状态，自动重连
3. **简易配置**：支持配置文件、命令行参数、环境变量多种方式
4. **跨平台支持**：支持 Windows、Linux、macOS

## 项目目标

- 提供简单易用的校园网登录工具
- 省去复杂的加密逻辑实现，直接通过浏览器自动化完成登录
- 支持后台运行，自动保持网络连接

## 目标用户

- 北航校园网用户
- 需要稳定网络连接的服务器/设备
- 不想手动重复登录的用户

## 非目标

- 不实现复杂的加密协议逆向
- 不提供 GUI 界面
- 不支持其他学校的认证系统（但架构可扩展）

## 成功标准

- 能够成功完成校园网登录
- 能够稳定保持网络连接
- 安装和使用简单便捷
